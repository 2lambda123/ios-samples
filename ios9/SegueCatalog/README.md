# SegueCatalog

Sample demonstrates how to combine `UIStoryboardSegue` subclasses with `Transition Delegates` and `Adaptivity`, and how to use unwind segues

## Build Requirements

Building this sample requires Xcode 7.0 and iOS 9.0 SDK
